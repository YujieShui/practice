# 安卓入门学习

感谢《第一行代码》这本书，让我对安卓有了一个大致的了解，快速地入了一个门。从 8 月 3 日开始学习——8 月 13 日，整好10天看完了这本书，其中大部分代码都自己实现了一遍。里面一个个设计精巧的案例环环相扣，每一个知识点都恰好在你能接受的范围之内。

利用这几天每天晚上几个小时时间学习了安卓真的是蛮充实的。但这显然只是一个开始，现在只是了解了最基础的应用，了解了大致的内容，还需要勤加练习，多多总结。
